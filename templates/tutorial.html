Arya's HTML Tutorial

First, what is HTML?

---------------- > HTML Is the standard language for creating and structuring websites. It's generally front-end coding, and deals with the design aspect of a website rather than the logic and back-end part.

---------------- > CSS deals with the literal design for elements inputed in HTML (I.E color, positioning, etc). This is still considered front-end code as it's still dealing with the visual aspect of a website.

Now Let's learn about some HTML ELements!

1. '!DOCTYPE html' states that the current document being edited is an HTML document, and ensures that you're within the bounds of modern web standards (preventing 'quirks mode').

2. 'head' contains what's called the non-visible "meta information," AKA information about the HTML Document itself (i.e page title, links to CSS, etc)

3. 'html' is the root element of the website. Pretty much an organizational feature that encapsulates all the elements of your website.

4. 'title' is pretty self-explanatory, but more broadly, it represents one of the ways you can format tax on your website.

5. 'body' contains everything that is visible on the page. is different because it also contains metadata.

6. 'h1' represents a way to format text, in this case being a header, and is the biggest version on the scale of headers (h1-h6).

7. 'p' also represents a way to format text, in this case being basic text AKA a paragraph element.
